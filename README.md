# Read before using.

This is a simple tool that lets you turn your magic items into cards special card frames. All cards will be arranged on an A4 template as a single image. You’ll be able to check if everything is positioned correctly, then save the file and print it for use in your games.

App allows you to enter any text you want in fields such as bulk, usage, etc., be sure to double-check that everything looks right before printing.

The reasoning behind that is an idea to make the app as flexible as possible so you can adapt it to your own needs. You never know when someone might need to put “really heavy” as the bulk value.

This app is desktop only and usage browser's local storage.
